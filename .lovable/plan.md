
# Обновление главной страницы: виджеты развития + курсы

## Концепция

Сейчас главная -- это дашборд тренажера (прогресс навыков, радар, задания). Нужно добавить виджеты, отражающие общее развитие пользователя в рамках LMS:

```text
+-------------------------------+-------------------+
| Приветствие + общий прогресс  | Радар навыков     |
+-------------------------------+-------------------+
| Мои курсы (горизонтальная     | Streak / дни      |
|  лента текущих курсов)        | обучения          |
+-------------------------------+-------------------+
| Следующее   | Мои задания     | Каталог курсов    |
| задание     |                 |                   |
+-------------+-----------------+-------------------+
```

## Новые виджеты

### 1. "Мои курсы" -- карточки текущих курсов
- Горизонтальная лента записанных курсов (из `useUserCourses`)
- Каждая карточка: название, прогресс-бар, кнопка "Продолжить"
- Если курсов нет -- кнопка "Перейти в каталог"
- На десктопе: `lg:grid-cols-2` или `lg:grid-cols-3`

### 2. "Streak / Активность" -- дни обучения
- Виджет "Вы занимались X дней на этой неделе"
- 7 точек/кружков (Пн-Вс), активные подсвечены
- Данные пока mock (статичные), т.к. таблицы activity log нет
- Градиентная карточка, мотивационный стиль

### 3. "Каталог курсов" -- быстрый доступ
- Заменяет текущую карточку "Записи вебинаров"
- Показывает количество доступных курсов
- Ведёт на `/catalog`

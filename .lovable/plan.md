
# План: Исправление выхода иконок за границы на мобильных устройствах

## Проблема
На мобильных устройствах иконки навыков вокруг радар-чарта выходят за границы контейнера. Это происходит из-за фиксированного радиуса позиционирования иконок (`iconRadius = 135`) и фиксированного размера иконок (48x48px).

## Решение
Сделать радиус и размер иконок адаптивными с помощью хука `useIsMobile`.

## Изменения

### Файл: `src/pages/Index.tsx`

1. **Добавить импорт хука**:
   ```tsx
   import { useIsMobile } from "@/hooks/use-mobile";
   ```

2. **Использовать хук в компоненте**:
   ```tsx
   const isMobile = useIsMobile();
   ```

3. **Адаптировать параметры в RadarChart**:
   - Уменьшить `iconRadius` с 135 до 100 на мобильных
   - Уменьшить размер иконок с 48px до 40px на мобильных
   - Уменьшить высоту контейнера с 320px до 280px на мобильных

4. **Обновить код рендеринга иконок**:
   ```tsx
   const iconRadius = isMobile ? 100 : 135;
   const iconSize = isMobile ? 40 : 48;
   
   <foreignObject 
     x={-(iconSize / 2)} 
     y={-(iconSize / 2)} 
     width={iconSize} 
     height={iconSize}
   >
     <div className={`${isMobile ? 'w-10 h-10' : 'w-12 h-12'} rounded-2xl ...`}>
   ```

## Технические детали

| Параметр | До (desktop) | После (mobile) |
|----------|--------------|----------------|
| Высота контейнера | 320px | 280px |
| Радиус иконок | 135px | 100px |
| Размер иконок | 48x48px | 40x40px |

Это обеспечит, что иконки останутся внутри границ карточки на экранах шириной от 320px.

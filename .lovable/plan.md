

# Прогресс: Карта компетенций + Прогресс по курсам

## Концепция

Раздел "Мой прогресс" получает табы-слайдер для переключения между двумя видами:

1. **Карта AI-компетенций** (по умолчанию) -- визуальная карта из 5 базовых навыков (ядро) + 9 направлений с 3 уровнями каждое
2. **Прогресс по курсам** -- текущий контент (радар + список навыков тренажера + курсы пользователя)

```text
+---------------------------------------------------+
| < Назад    Мой прогресс               [Settings]  |
+---------------------------------------------------+
| [ Карта компетенций ]  [ Мои курсы ]               |
+---------------------------------------------------+
|                                                     |
|  (содержимое активного таба)                        |
|                                                     |
+---------------------------------------------------+
```

## Карта AI-компетенций -- первый таб

### Структура данных (статические, без БД)

Вся карта описывается в новом файле `src/data/competencyMap.ts`:

- **Базовое ядро**: 5 навыков (формулировка запросов, проверка AI, контекст, безопасность, обучаемость)
- **9 направлений**: Текст, Данные, Визуал, Планирование, AI-агенты, AI-решения, Human Skills, Этика, Стратегия
- Каждое направление: 3 уровня с названием, описанием и списком навыков

### UI компонент `src/components/progress/CompetencyMap.tsx`

**Секция "Базовое ядро":**
- Карточка с градиентом и иконкой "ядра"
- 5 навыков ядра как чипы/бейджи внутри карточки
- Каждый чип кликабельный -- раскрывает описание

**Секция "9 направлений":**
- Grid карточек `lg:grid-cols-3` (на мобильном -- вертикальный список)
- Каждая карточка направления:
  - Иконка + название + номер
  - 3 уровня как шаги (L1 / L2 / L3) -- визуально как progress steps
  - Краткое описание каждого уровня
  - Клик по карточке раскрывает подробности (список навыков) через Sheet или аккордеон

**Визуальный стиль:**
- Каждое направление имеет свой цвет (9 цветов)
- Уровни показаны как 3 точки/сегмента: пустые (не освоено), заполненные (освоено)
- Пока все уровни "пустые" (нет данных о прогрессе) -- просто информационная карта

### UI компонент `src/components/progress/DirectionCard.tsx`

Карточка одного направления:
- Заголовок с иконкой и номером
- 3 уровня как строки: "L1: Переписки и письма", "L2: Документы и отчёты", "L3: Контент и публикации"
- Клик раскрывает Sheet с полным описанием направления, навыками и источниками

### UI компонент `src/components/progress/CoreSkillsCard.tsx`

Карточка базового ядра:
- Градиентный фон
- 5 навыков с иконками
- Кликабельные -- раскрывают "почему критично" и "что нужно уметь"

## Второй таб -- "Мои курсы"

### UI компонент `src/components/progress/CourseProgress.tsx`

Содержимое текущей страницы MyProgress (радар + список навыков) + добавляется секция с прогрессом по записанным курсам:

- Радар-чарт навыков тренажера (существующий)
- Список навыков тренажера (существующий)
- Секция "Курсы" -- карточки записанных курсов с прогресс-барами (из `useUserCourses`)

## Обновление MyProgress.tsx

Текущее содержимое оборачивается в `Tabs` (Radix):

```text
<Tabs defaultValue="competency-map">
  <TabsList>
    <TabsTrigger value="competency-map">Карта компетенций</TabsTrigger>
    <TabsTrigger value="courses">Мои курсы</TabsTrigger>
  </TabsList>
  <TabsContent value="competency-map">
    <CompetencyMap />
  </TabsContent>
  <TabsContent value="courses">
    <CourseProgress />
  </TabsContent>
</Tabs>
```

## Траектории развития

Внизу карты компетенций -- секция "Траектории":
- 5 готовых профилей (Специалист, Менеджер, Руководитель, Power User, Финансист)
- Каждый профиль -- карточка с описанием рекомендованных направлений
- Клик подсвечивает рекомендованные направления на карте

## Техническая реализация

### Новые файлы

| Файл | Назначение |
|------|-----------|
| `src/data/competencyMap.ts` | Статические данные карты: ядро, 9 направлений, 3 уровня, навыки, траектории |
| `src/components/progress/CompetencyMap.tsx` | Основной компонент карты компетенций |
| `src/components/progress/CoreSkillsCard.tsx` | Карточка базового ядра (5 навыков) |
| `src/components/progress/DirectionCard.tsx` | Карточка направления (иконка, 3 уровня, описание) |
| `src/components/progress/TrajectorySection.tsx` | Секция траекторий развития |
| `src/components/progress/CourseProgress.tsx` | Текущий контент прогресса (радар + навыки + курсы) |

### Модифицируемые файлы

| Файл | Изменение |
|------|-----------|
| `src/pages/MyProgress.tsx` | Добавить Tabs, вынести текущий контент в CourseProgress, добавить CompetencyMap |

### Иконки направлений

Используем Lucide-иконки (уже установлены):
1. Текст -- `FileText`
2. Данные -- `BarChart3`
3. Визуал -- `Image`
4. Планирование -- `Calendar`
5. AI-агенты -- `Bot`
6. AI-решения -- `Wrench`
7. Human Skills -- `Heart`
8. Этика -- `Shield`
9. Стратегия -- `Target`

### Порядок реализации

1. Создать `src/data/competencyMap.ts` с полными данными карты
2. Создать компоненты `CoreSkillsCard`, `DirectionCard`, `TrajectorySection`
3. Создать `CompetencyMap` (собирает всё вместе)
4. Создать `CourseProgress` (вынести текущий контент + добавить курсы)
5. Обновить `MyProgress.tsx` -- обернуть в Tabs
